Sobre USB AI

Proposito del Proyecto

El propósito de este proyecto es crear un asistente educativo inteligente, capaz de ayudar a estudiantes y educadores con sus necesidades académicas. USB AI está diseñado para proporcionar respuestas claras y concisas a consultas educativas, haciendo que el aprendizaje sea más accesible e interactivo. Al aprovechar tecnologías de inteligencia artificial avanzadas, este proyecto busca cerrar la brecha entre la educación tradicional y la tecnología moderna. Además, esta iniciativa se alinea con el Objetivo de Desarrollo Sostenible (ODS) 4: Educación de Calidad, promoviendo el acceso equitativo al conocimiento a través de herramientas innovadoras.

¿Comó lo hicimos?

Este proyecto fue desarrollado utilizando tecnologías web modernas como Spring Boot para el backend y Tailwind CSS para el frontend. El chatbot aprovecha modelos de inteligencia artificial avanzados para brindar asistencia educativa de manera eficiente. Entre las técnicas empleadas, se utilizó un algoritmo de Zero-Shot Classification para clasificar las preguntas del usuario sin necesidad de entrenamiento específico previo, lo que permite una mayor versatilidad en las respuestas. Además, se integró un modelo local mediante Ollama, lo que garantiza mayor privacidad, bajo tiempo de respuesta y la posibilidad de operar sin conexión a internet.
